Field-dependent Kernel Spline
-----------------------------

The following utlity functions are used for a quasi-linear scaling implementation
of non-local kinetic functionals which have a single-point density dependent kernel, such as
the :ref:`Huang-Carter <hc>` and :ref:`revised Huang-Carter <revhc>` functionals.
