Welcome to PROFESS-AD's documentation!
=======================================

**PROFESS-AD** is PyTorch-based auto-differentiable plane-wave orbital-free density functional theory (OFDFT) code. 
For a brief description of the package, see the :doc:`about` page. For installation details, see the :doc:`install` page.
Details of the various functionalities of PROFESS-AD can be found in the :doc:`user_guide` page while
examples are presented in the :doc:`examples` page.
